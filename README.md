## Barber PRO - Backend
The objective of this project is to create a backend for a for a barbershop system.

## Technology 

Here are the technologies used in this project.

* PostgreSQL
* Prisma
* Node Js
* Json Web Token
* Bcrypt
* Cors
* Express
